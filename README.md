# 🛡️ Tria Sigorta — Insurance Agency Website

A professional, modern website built for **Tria Sigorta**, an insurance agency based in İzmir, Turkey specializing in life insurance and retirement plans.

🔗 **Live:** [triasigorta.com](https://triasigorta.com)

---

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Scroll-triggered animations
- Interactive phone call popup modal
- Google Maps integration
- Customer reviews section
- Service showcase with hover effects
- SEO optimized with meta tags

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript** — Intersection Observer, DOM manipulation
- **Google Fonts** — Outfit + Sora
- **Hosting** — Netlify
- **Domain** — GoDaddy

## 📸 Sections

- Hero with statistics
- Services (Life & Retirement, Auto, Health, Home, DASK, Traffic)
- Image showcase gallery
- About the company
- Customer reviews (Google 5.0 ⭐)
- Contact with Google Maps
- Call-to-action with phone popup

## 🚀 Deployment

The site is deployed on Netlify with a custom domain configured via GoDaddy nameservers.
